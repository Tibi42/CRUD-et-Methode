Les cookies et leurs expirations

Tips sur les dates en javascript

Fullstack Developer | Symfony & React

Les cookies sont des petits fichiers de données stockés sur le navigateur de l'utilisateur. Ils sont utilisés pour garder une trace des informations de session, des préférences utilisateur, et d'autres données pertinentes.

En JavaScript, la gestion des cookies se fait principalement via l'objet document.cookie. Il est important de comprendre comment définir, lire et supprimer les cookies, ainsi que leur date d'expiration.

Gestion des cookies en JavaScript

Pour créer un cookie, vous pouvez utiliser la syntaxe suivante :

document.cookie = "nom=valeur; expires=date; path=chemin";

Voici un exemple de création d'un cookie avec une date d'expiration :

document.cookie = "username=JeanQui; expires=Fri, 31 Dec 2025 23:59:59 GMT; path=/";

Dans cet exemple, le cookie "username" est créé avec une valeur de "JeanQui" et expirera le 31 décembre 2025 à 23h59 GMT.

Lire un cookie

Pour lire un cookie, vous pouvez utiliser la propriété document.cookie. Cela renverra une chaîne contenant tous les cookies disponibles pour le domaine actuel.

Voici un exemple de lecture d'un cookie :

let cookies = document.cookie.split('; ');
    for (let i = 0; i < cookies.length; i++) {
        let cookie = cookies[i].split('=');
        if (cookie[0] === 'username') {
            console.log('Username:', cookie[1]);
        }
    }

Dans cet exemple, nous parcourons tous les cookies et affichons la valeur du cookie "username".

Supprimer un cookie

Pour supprimer un cookie, vous devez le définir avec une date d'expiration dans le passé :

document.cookie = "username=; expires=Thu, 01 Jan 1970 00:00:00 GMT; path=/";

Dans cet exemple, le cookie "username" est supprimé en le définissant avec une date d'expiration dans le passé.

Tips pour manipuler les dates en JavaScript

Les dates en JavaScript sont manipulées à l'aide de l'objet Date. Voici quelques astuces utiles :

  • Créer une date actuelle : let now = new Date();
  • Créer une date spécifique : let specificDate = new Date('2025-12-31T23:59:59');
  • Obtenir l'année : now.getFullYear();
  • Obtenir le mois (0-11) : now.getMonth();
  • Obtenir le jour du mois : now.getDate();
  • Formater une date en chaîne : now.toISOString();

Créer et supprimer un cookie avec une date

Voici un exemple de création et de suppression d'un cookie :


    // Pseudo-code pour créer un cookie avec une date d'expiration
    document.cookie = "user=JohnDoe; expires=Fri, 31 Dec 2025 23:59:59 GMT; path=/";
    
    // Pseudo-code pour supprimer un cookie en définissant une date d'expiration passée
    document.cookie = "user=; expires=Thu, 01 Jan 1970 00:00:00 GMT; path=/";

Dans cet exemple :

  • Le cookie user est créé avec une valeur de JohnDoe et une date d'expiration au 31 décembre 2025.
  • Pour supprimer le cookie, on le redéfinit avec une date d'expiration dans le passé.
